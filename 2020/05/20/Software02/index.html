Windows 软件推荐（二）

发表于 2020-05-20 更新于 2020-05-21 分类于 Computer Skills 阅读次数：

本文继续推荐软件，包括 Typora、CCleaner、IDM、Adobe Acrobat Pro DC、PotPlayer、foobar2000。

Markdown 编辑器：Typora

Typora 界面只有菜单栏若干几个选项和一个空白的输入区域，但这个简约的软件所支持的功能特别多。

  • 支持实时预览 Markdown 格式效果，同时还保留了「源代码模式」，快捷键 Ctrl + / 随时切换源码与预览（它的 Markdown 输入手感真是一流！）；
  • 配合 Pandoc 你可以轻松地将文件导出为 PDF、HTML、Word、移动端长图，以及许多学术相关的文件格式；
  • 支持 LaTeX 等公式符号、表格、代码、图片外链。基本你能想到的文稿内容它都支持并可以实时预览。

少数派: 我想开门见山地直言，Typora 是我在桌面端最喜爱的生产力应用。它的功能之强大、设计之冷静、体验之美妙、理念之先进，我认为值得所有笔记应用厂商学习。

清理注册表：CCleaner

CCleaner 可用于清理系统注册表、优化和隐私保护工具，可用于 Windows 电脑清理。比较专业，无广告，不流氓！

下载工具：IDM

Internet Download Manager (IDM) 是一款老牌下载器，功能强大，它还提供了许多贴心功能。它支持自动捕获链接、静默下载、批量下载、站点抓取等功能。

PDF 阅读编辑：Adobe Acrobat Pro DC

众所周知， Adobe 创建 PDF 这个电子文档格式，是为了让用户可以不因操作系统的语言、字体和显示设备的不同而改变文件原貌。正是因为 PDF 格式的这种其他格式文件无可比拟的特性，才使得 PDF 如此受欢迎。

Adobe Acrobat Pro DC 算是最为全面的 PDF 软件了，好用自不必说，就是有点大。

本地视频播放器：PotPlayer

PotPlayer 可以说是当今 Windows 平台上功能最强大的多媒体播放器（没有之一），无广告，不流氓，界面简洁，纯粹为播放而生。它既可以完美胜任普通用户的一般播放用途，也可以满足高级用户的特别需求与定制。

  • 功能强大，设置齐全（甚至多到有点复杂）；
  • 纯粹的本地播放器，体验好，没广告，不流氓；
  • 性能强劲，播放效果一流（还可配合插件增强）支持视频/音频格式全面；
  • 自定义选项丰富，可按喜好自由设置皮肤丰富，界面精美；
  • 快捷键非常方便
    • 播放/晢停：空格
    • 音量：鼠标滚轮，上/下方向键前进/后退：左/右方向键
    • 全屏/还原： Enter
    • 倍速：C-加速，X-减速，Z-复位
    • 上/下一帧：D/F
    • Ctrl+E：截图并保存，存储位置可在选项中选择
    • At+C：视频、GIF录制

曾经用过本地视频播放器 MPC-BE，QQ 影音，直到播放十几个 G 大小的电影时才认识到 PotPlayer 的强大。在其他播放器卡的不能播放时，PotPlayer 依然很流畅。

音乐播放器：foobar2000

foobar2000 是一款免费软件，是 Windows 平台下的高级音频播放器。它不注重外表的修饰，更在乎音质与实用功能上的完美追求。foobar2000 功能的发挥需要几个插件 参考。

foobar2000 主要用来听本地无损音乐（包括 wav、flac、ape 等格式），我主要用它来听古典音乐。
