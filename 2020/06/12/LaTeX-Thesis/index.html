SHNU 毕业论文 LaTeX 模板

发表于 2020-06-12 更新于 2023-12-10 分类于 LaTeX 阅读次数：

上海师范大学研究生毕业论文 LaTeX 模板

LaTeX 模板及样例在 GitHub 仓库：shnuthesis，可以直接下载。

注意： 本模板需要安装 texlive 发行版，并且需要 XeLaTeX 编译运行！

shnuthesis 文档类可指定的选项包括

  • master 硕士学位论文，不能省略
  • doctor 博士学位论文，不能省略
  • arts 文科学位论文, 默认缺省为理科
  • print 用于打印, 封面等生成空白页
  • 提交给图书馆的电子版，不要选 print

本模板为上海师范大学非官方的研究生学位论文 LaTeX 模板，参考了《上海师范大学研究生学位论文规范》，文档类为 shnuthesis.cls，示例文件 example.tex 给出了使用的指南和要求，也展示了论文排版中常用的例子，包括公式、定理、表格、插图、参考文献等。

电脑本地使用

本模板需要安装 texlive 发行版，并且需要 XeLaTeX 编译运行！如需安装 texlive，可以参考：LaTeX 之软件安装与配置 和 视频教程，如果之前安装了 CTEX 套装，安装之前最好把它卸载干净。

本地测试环境：Windows 10 安装 texlive 发行版。

某些 texlive 版本可能需要声明使用 Windows 字体 (fontset = windows)，才能调用通常的宋体、楷体和黑体。

在线使用

在线 LaTeX 编辑可以使用 TeXPage，设置字体 (fontset=windows) 后符合学校要求，推荐使用。

如果在线使用 Overleaf，设置字体 (fontset=ubuntu) 后可运行，但字体不符合学校要求，不推荐使用。

如有不足之处，欢迎提出意见或建议。
