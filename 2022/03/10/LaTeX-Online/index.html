shnuthesis 在线使用教程

发表于 2022-03-10 更新于 2024-03-03 分类于 LaTeX 阅读次数：

上海师范大学研究生毕业论文 LaTeX 模板在线使用教程

使用国内 LaTeX 在线编辑平台 TeXPage，能够满足研究生毕业论文的要求。

本模板推荐安装 texlive 发行版，并且需要 XeLaTeX 编译运行！参考模板说明。

不支持 CTEX 套装，直接使用会出现很多问题。

如果条件不允许，或者临时使用，可以使用在线 LaTeX 编辑平台。(有些科研人员把在线 LaTeX 平台作为主力的情况也是有的，在线平台备份做的很好)。

本模板可以使用到在线 LaTeX 编辑网站 Overleaf 上使用，设置字体 (fontset = ubuntu) 可以使用，这样使用的问题是汉字就不是通常的宋体、楷体和黑体，而是开源的 Fandol 字体，不符合学校对毕业论文的要求。不推荐使用

注：对于英文论文，推荐使用 Overleaf，但在国内的网络环境下不流畅。

推荐使用 TeXPage 平台

TeXPage 网站 https://www.texpage.com 最好使用火狐浏览器或 Google 浏览器登录。

模板最好从 Github 下载最新版 https://github.com/andy123t/shnuthesis， 注册 TeXPage 账号，然后按要求将模板上传到网站 (空项目上传文件或上传 zip 压缩文件)。

用户可以在 TeXPage 模板库搜索上海师范大学研究生毕业论文 LaTeX 模板，使用此模板新建项目。

LaTeX 编辑器选择一个 TeX Live 版本使用 XeLaTeX 编译，主文件为 example.tex

注意：设置使用 Windows 字体 (通常的宋体、楷体、黑体以及英文字体) ：

选择菜单栏『更多』可以直接下载源文件或编译完成的 PDF 文件。
